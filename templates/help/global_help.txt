
dirforge
====================================================================================================================

Create standardized directory structures per DirForge Constitution

Constitution Version: v1.0.16


Usage
-----

  dirforge <command> [options]


Commands
--------

  init <world-type>                                                                   Create new project structure


Global Options
--------------

  --help, -h                                                                                Show this help message
  --version                                                                               Show version information
  --dry-run, --preview                                               Preview changes without writing to filesystem
  --json                                                    Output machine-readable JSON plan (requires --dry-run)


World Types
-----------

  research                                                         Academic research projects with data management
  lecture                                                               Educational content with grading workflows
  coding                                                                             Software development projects
  journal                                         Journal-related activities: submissions, reviews, editorial work
  office                                                                     Administrative and business documents
  private                                                                  Personal projects with privacy controls


Examples
--------

Common Usage:)

  # Create research project with interactive prompts
  dirforge init research --title "Thermal Analysis"
  
  # Create lecture project structure
  dirforge init lecture --name "Digital Rock Physics"
  
  # Preview project creation with JSON output
  dirforge --dry-run init research --title "Test" | jq .


More Information
Use 'dirforge <command> --help' for detailed command information.
See constitution.md for governance and naming rules.

Constitution compliance ensures consistent project structures across teams and organizations.
